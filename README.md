# Install

Use either of these repositories:

```

    jcenter()
}
```

Or:

```

    maven {
        url 'https://dl.bintray.com/novacrypto/<PROJECT>/'
    }
}
```

Add dependency:

```

    compile 'io.github.novacrypto:<PROJECT>:0.1.2@jar'
}

```
